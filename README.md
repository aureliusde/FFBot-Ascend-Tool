# FFBot Ascension Tool

A clean, offline browser tool to help you decide **when to ascend** and **what stats to aim for** in FFBot.

## Live Demo

**[Open FFBot Ascension Tool →](https://aureliusde.github.io/FFBot-Ascend-Tool/ffbot_ascend_tool.html)**

## Features

- Calculates recommended target stats (2.0x, 2.2x, 2.5x multipliers)
- Adjusts advice based on your current **NG+** level (more conservative at high NG+)
- Role-specific recommendations (**Damage Dealer**, **Tank**, **Healer**)
- Gives clear guidance on whether you should ascend now or keep pushing

## How to Use

1. Click the link above to open the tool
2. Enter your current **NG+**
3. Select your role
4. Input your current stats (ATK, MAG, SPI, HP)
5. Click **"Calculate Next Targets"**

The tool will instantly show you:
- Recommended multiplier range
- Target stats for different multipliers
- Personalized advice

## Technical Info

- Single HTML file (works completely offline)
- No data is collected or stored
- Built with pure HTML, CSS, and JavaScript
